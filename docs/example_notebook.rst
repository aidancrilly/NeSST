Example Notebook
================

A Jupyter notebook guide to NeSST is available in the ``example/`` directory of the
`GitHub repository <https://github.com/aidancrilly/NeSST>`_.

The notebook covers:

- Setting up energy grids and scattering matrices
- Computing primary neutron spectra (DT, DD, TT)
- Calculating singly-scattered neutron spectra
- Applying areal density asymmetry effects
- Using ion kinematics corrections
- Synthetic neutron time-of-flight detector signals

You can `view the notebook interactively on nbviewer
<https://nbviewer.org/github/aidancrilly/NeSST/blob/master/example/NeSST%20Guide.ipynb>`_.
